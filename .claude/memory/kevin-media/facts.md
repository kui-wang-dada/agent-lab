## 工艺关键点（重要）

### 文案 ≈ 字幕原则
- Kevin 录视频时基本按 `01-script.md` 念（允许少量即兴）
- 字幕**文字**用文案原文（100% 正确）
- 字幕**时间戳**由 Whisper 词级识别 + 强制对齐得出
- 摆脱"识别 → 人工校对" 循环

### Pipeline 自动化
- Docker Compose 一键剪辑（去静音、烧字幕、烧图、混音、封面）
- **直接拿成片去发，不返剪映**
- 调风格用 `PREVIEW=60` 只剪前 60 秒（2-3 分钟出预览）

### 预览模式必须每次重跑（2026-05-09 拍板）
- `PREVIEW=60` 时自动追加 `--force`，不读任何缓存
- 理由：预览本身就是为了看新参数效果，缓存就违背初衷

### 选题工作流（Kevin 2026-04-18 自己拍板）
1. 周日 21:00 调度任务 `media-weekly-topics` 跑：先刷新 `kevin-voice.md`，再按主题感知抓素材
2. **不替 Kevin 选题**：Kevin 周一从素材库 / 自己脑子里定，可能根本不从素材库选——素材库只是参考
3. 文案 Kevin 先写大概 → agent 帮完善 → Kevin 收图 → 录视频 → pipeline 出成片
4. 信息源：B 站 / YouTube / 公众号 / 抖音；**抖音目前没有 MCP 拿不到视频级链接，保留搜索词级**（Kevin 暂不投资装第三方）

### Pipeline 边界（Kevin 2026-05-04 / 2026-05-10 反复拍板）
- **原则**：不要给 Kevin 加工作量；自动化能做的全做掉，让效果变好
- **Kevin 明确拒绝**的功能（不要再推荐）：
  - 竖屏短版自动切片（不做竖屏视频）
  - 章节时间码（平台会自己处理）

- **Kevin 接受**的：BGM 自动配 + 自动 ducking、自动片头片尾、音频处理链、多尺寸封面（横/竖/方）
