# kevin-assistant 的工作经验积累

> Agent 在工作中学到的可复用经验。任何完成的任务，若有"下次类似情况怎么做更好"的洞察，追加到这里。

## 格式

```markdown
## YYYY-MM-DD — <一句话主题>
**发生了什么**：本次任务做了什么
**学到什么**：什么有效 / 什么失败 / 为什么
**下次怎么做**：具体可执行的改进
**适用场景**：什么时候用得上这条经验
```
